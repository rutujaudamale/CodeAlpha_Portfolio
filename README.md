# Rutuja Udamale — Portfolio Site

A personal portfolio website built for the **CodeAlpha Frontend Development Internship**, showcasing my work as a Data Analyst and Web Developer.

🔗 **Live Demo:** _add your GitHub Pages link here after deploying_

## Features

- Responsive single-page design with smooth scroll navigation
- Typewriter-style animated headline switching between "Data Analyst" and "Web Developer"
- Animated hero dashboard graphic with scroll-triggered reveal effects
- Hover interactions on project and skill cards, including a cursor-following glow effect
- Toggle-based project section — switch between Data Analysis and Web Development projects
- Downloadable resume (PDF)
- Fully responsive down to mobile screen sizes

## Sections

- **About** — background and current focus
- **Skills** — Data Analysis (Python, SQL, Pandas, NumPy, Matplotlib, Excel, Power BI, MySQL) and Web Development (HTML, CSS, JavaScript, Git, Netlify)
- **Projects** — data analysis projects (EDA, forecasting, dashboards) and web development projects
- **Resume** — direct PDF download
- **Contact** — email, LinkedIn, GitHub

## Built With

- HTML5
- CSS3 (custom properties, Grid, Flexbox, animations)
- Vanilla JavaScript (IntersectionObserver for scroll reveals, typewriter effect)
- Google Fonts — Sora, IBM Plex Mono, Work Sans

## Running Locally

Clone the repo and open the HTML file directly in a browser — no build step or dependencies required.

```bash
git clone https://github.com/rutujaudamale/CodeAlpha_Portfolio.git
cd CodeAlpha_Portfolio
```

Then open `portfolio.html` (or `index.html`) in your browser.

## Deploying with GitHub Pages

1. Rename `portfolio.html` to `index.html` (or set it as the entry point in your Pages settings)
2. Go to **Settings → Pages** in the repository
3. Set the source branch to `main` and folder to `/ (root)`
4. Your site will be live at `https://rutujaudamale.github.io/CodeAlpha_Portfolio/`
